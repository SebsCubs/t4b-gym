# Environment Setup

This project uses a Conda environment. To set up the environment:

1. Install Miniconda or Anaconda if you haven't already
2. Create and activate the environment:
   ```bash
   conda env create -f environment.yml
   conda activate t4b_env
   ```
3. Install additional pip requirements:
   ```bash
   pip install -r requirements.txt
   ```
4. Install twin4build by going to the main repository and completing all requirements: [Twin4Build](https://github.com/JBjoernskov/Twin4Build)
